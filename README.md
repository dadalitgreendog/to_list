# to_list

一个像素风老虎机玩具，实则是一个to do抽取网页。

## 项目说明

老虎机可以用于添加多个to do list，并在每个list里面添加多个to do，然后通过等概率随机抽取的方式决定接下来做什么。
当前版本是一个纯前端静态页面，主要文件是 `index.html`。

## 当前功能

- 新建、切换、重命名、删除 list。注：给list取一个有趣的名字吧~list的名字会同步在顶部标题的to _ list的_位置哦~
- 为当前 list 添加和删除 to do。
- 抽中后自动禁用该项，避免重复抽取，也可以手动恢复。
- 保存抽取历史。
- 使用浏览器本地存储保留数据。

## 本地使用

直接在浏览器中打开 `index.html` 即可。
